# Bachué Analytics Hub

Este es el portal oficial de **Bachue Lab SAS** enfocado en la divulgación, cursos y herramientas interactivas de estadística y analítica de datos. Hospedado en GitHub Pages de forma estática y ultrarrápida.

## 📁 Estructura del Proyecto

```text
bachue-analytics/
├── index.html              # Landing Page principal
├── hub.html                # Centro de mando (Hub de calculadoras)
├── assets/                 # Recursos compartidos
│   ├── css/
│   │   └── main.css        # Estilos generales para toda la marca
│   ├── js/
│   │   └── main.js         # Lógica de interactividad general
│   └── img/                # Logos y fotos de perfil
└── calculadoras/           # Carpeta dedicada a las herramientas
    ├── descriptiva.html
    ├── probabilidad.html

    ├── hipotesis.html
    ├── experimentos.html
    └── lineal.html
